# MiMo Lite GUI

A lightweight local browser GUI for MiMo Code/OpenCode-compatible APIs.

## Download

Go to Releases and download `mimo-lite-gui.zip`.

## Requirements

- Node.js 18+
- MiMo CLI available as `mimo`, or set `MIMO_BIN`

## Run

```bash
unzip mimo-lite-gui.zip
cd mimo-lite-gui
npm install
MIMO_PROJECT_DIR="/path/to/your/project" npm start
